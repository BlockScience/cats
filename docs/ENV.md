### Manage CATs' Virtual Environment:
##### 0. Create `venv`:
```bash
# CATs working directory
cd <CATs parent directory>/cats-research
python -m venv ./venv
```
#### 1. Activate/Deactivate Virtual Environment
**Activate `venv`:**
```bash
source ./venv/bin/activate
# (venv) $
```
**Deactivate `venv` (Optional):**
```bash
deactivate
# $
```
